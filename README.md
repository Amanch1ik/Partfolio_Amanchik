# PANEL-s YESS-Go

Админ-панель и панель партнера для системы лояльности YESS! GO.

## Структура проекта

```
PANEL-s_YESS-Go/
├── admin-panel/          # Админ-панель (React + TypeScript)
├── partner-panel/        # Панель партнёра (React + TypeScript)
└── panels-ts-v2/         # Обновленные панели
```

## Быстрый старт

### Admin Panel

```bash
cd admin-panel
npm install
npm run dev
```

### Partner Panel

```bash
cd partner-panel
npm install
npm run dev
```

## Технологии

- React 18 + TypeScript
- Vite
- Ant Design
- React Query
